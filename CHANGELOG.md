# @opensourceframework/next-json-ld

## 0.1.0

### Minor Changes

- Initial release of @opensourceframework/next-json-ld - JSON-LD structured data helpers for Next.js SEO.

  Features:
  - Type-safe Schema.org JSON-LD generation
  - Organization, Service, FAQ, Breadcrumb, Review, Product, Article, and Event schemas
  - Full TypeScript support with comprehensive interfaces
  - Next.js App Router and Pages Router support
  - Zero dependencies, tree-shakeable
  - Schema merging for multiple structured data types
  - Full test coverage

- Initial release of new open-source packages extracted from Next.js projects

## 0.0.1

### Initial Release

- Add `createOrganizationSchema` for local business and organization structured data
- Add `createServiceSchema` for service offerings
- Add `createFAQSchema` for FAQ page markup
- Add `createBreadcrumbSchema` for navigation breadcrumbs
- Add `createReviewSchema` for reviews and testimonials
- Add `createProductSchema` for e-commerce products
- Add `createArticleSchema` for blog posts and news articles
- Add `createEventSchema` for events
- Add `createJsonLdScript` helper for generating JSON-LD script content
- Add `mergeSchemas` utility for combining multiple schemas
- Full TypeScript support with comprehensive type definitions
- Zero dependencies
